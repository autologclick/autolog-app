יצירת אייקון Maskable ל-PWA

זה הכלי שיוצר אייקון מיוחד עבור התקנה כ-PWA על אנדרואיד. הוא מרפד את הלוגו על רקע לבן עם safe zone, כך שכשאנדרואיד חותך לעיגול — הלוגו לא נחתך.

איך זה ייראה כקובץ (512×512)

איך זה ייראה אחרי חיתוך עיגולי באנדרואיד

מה לעשות אחרי ההורדה:

1. שמור את הקובץ icon-maskable-512.png בתיקייה autolog-app/public/
2. פתח את autolog-app/public/manifest.json
3. מצא את הסוגריים המרובעים "icons": [ והוסף שם את הרשומה הבאה (לפני ]): 
,{
  "src": "/icon-maskable-512.png",
  "sizes": "512x512",
  "type": "image/png",
  "purpose": "maskable"
}
4. Deploy לפרודקשן + הסר את ה-PWA הקיים מהמסך הבית, ועיין שוב באתר → התקן מחדש. הפעם האייקון יתנהג יפה.
