================================================================================
                           INTEGRATION POINTS
================================================================================

MIDDLEWARE (src/middleware.ts)
  - Runs on every request
  - Extracts JWT from auth-token cookie
  - Validates JWT signature
  - Extracts user role
  - Checks route permissions
  - Enforces RBAC

API ROUTES
  - /api/auth/login: rate limited (5/min)
  - /api/auth/register: rate limited (3/min) + sanitized
  - /api/vehicles GET: rate limited (100/min)
  - /api/vehicles POST: rate limited (100/min) + sanitized

SECURITY HEADERS
  - Applied to all routes via next.config.js
  - No code changes needed in routes
  - Automatic with every response

RATE LIMITING
  - Tracks by IP for login/register
  - Tracks by user ID for API calls
  - In-memory storage (not persistent)
  - Auto-cleanup every 5 minutes

INPUT SANITIZATION
  - Removes HTML/script tags
  - Applied to string inputs
  - Ready to use in all routes
  - Already integrated in login/register/vehicles

OWNERSHIP VERIFICATION
  - Helper function available in api-helpers.ts
  - Ready for use in vehicle endpoints
  - Example: requireOwnership(userId, vehicle.userId)

A: Try logging in 6 times rapidly. 6th attempt returns 429.

A: 32+ characters with mixed case, numbers, and special characters.
